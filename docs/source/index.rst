wikidot.py ドキュメント
===================

Wikidotサイトとの対話を行うためのPythonライブラリ

このライブラリは、WikidotサイトのAPI操作を抽象化し、直感的なインターフェースを提供します。
ユーザー、サイト、ページなどのWikidotの主要要素にアクセスするための各種クラスが含まれています。
