インストール方法
==============

PyPIからのインストール
-----------------

wikidot.pyは、PyPIからインストールできます：

.. code-block:: bash

    pip install wikidot

開発版のインストール
--------------

最新の開発版をインストールするには、GitHubリポジトリから直接インストールします：

.. code-block:: bash

    pip install git+https://github.com/ukwhatn/wikidot.py.git@develop

必要条件
------

* Python 3.10以上
* httpx
* beautifulsoup4
* lxml
